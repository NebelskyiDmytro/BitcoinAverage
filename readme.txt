Bitcoin Price Analyzer

Configuration

apiUrl: Binance API URL.
apiSymbol: Symbol for Bitcoin in USD.
apiInterval: Interval for fetching data (e.g., '1d' for daily data).
apiLimit: Limit for fetching historical data.
csvFilePath: Path for the generated CSV file.
priceType: Choose between open, highest, lowest and close price types.

Check the generated CSV file bitcoin_prices.csv in the project directory.

Execute code: node bitcoinprice_average.js.
